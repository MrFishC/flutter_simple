+ flutter 3.x环境（windows），运行开源项目（flutter2.x）后，运行报错：your app to be migrated to the Android embedding
  v2
    + 处理方案
  ```
    <application
       android:name="com.flutter.hrl.flutter_hrlweibo">
    </application>
     将以上配置修改为以下的样子即可：
    <application
       android:name="${applicationName}">
    </application>
  ```
